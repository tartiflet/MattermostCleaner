# MattermostCleaner

De quoi effacer tes messages sur une instance mattermost.

Edites simplement le fichier MyServerConfig.py (sans trailing slash) avec les infos de ton instance Mattermost

/!\ pour une raison que je ne m'explique pas encore il faut parfois repasser le script plusieurs fois pour avoir la peau de tous les messages.
